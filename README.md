# Auto Create Invoice Design Tool

A single-file invoice design tool that runs directly in the browser.

## Features

- Editable company, client, invoice, and notes fields
- Dynamic line items with subtotal, discount, tax, and total calculations
- Currency selector
- Local browser saving through `localStorage`
- Print or save as PDF from the browser print dialog
- GitHub Pages ready

## Run Locally

Open `index.html` in a browser, or serve the folder with any static web server.
